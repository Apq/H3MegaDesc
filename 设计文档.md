# MegaDesc 设计文档

> 本文描述当前已实现的做法（对应 `MegaDesc.cpp` + `modules/*.inc.cpp` 实际代码）。

## 1. 功能概述

本插件扩展生物信息窗口：加高窗口、替换背景/按钮素材、扩大描述文本区。

触发方式：任意界面打开生物信息窗口（战斗/冒险/城镇）。

通过 exe patch 把窗口从原版 311 扩展到 487，并替换背景/按钮素材。

## 2. 窗口扩展与素材替换

### 2.1 exe patch

6 个地址（三界面×2），把原版 `push 0x137`(311) 改为 INI 配置高度（默认 487）：

| 界面  | patch 地址           |
| --- | ------------------ |
| 战斗  | 0x5F3F1B, 0x5F406B |
| 冒险  | 0x5F3728, 0x5F38CF |
| 城镇  | 0x5F45D8, 0x5F4712 |

### 2.2 PCX24→Pcx8 素材替换

插件目录 `pcx\` 下放置 24-bit / 3-plane PCX 素材（`bv_bgA.pcx`、`bv_okN.def` 等）。插件自行 RLE 解码 + 调色板量化为 `_Pcx8_*`，替换原控件的内部图片指针。不新建控件、不换 vtable。

### 2.3 描述文本控件

原版描述控件通过 `_DlgStaticText_::Create(0x5BC6A0)` 创建，参数由 `push` 指令压栈。`TextHeight > 127` 时不能用原版 `push imm8`（符号扩展问题），改为在 call 前 LoHook 改写栈参数（`x/y/w/h`），让创建函数收到真实 32 位值。

### 2.4 窗口 Y 吸附

扩展后的窗口可能超出 HD 渲染 buffer。在 `_Dlg_` 初始化入口 `0x41AFA0` LoHook 中按 HD 分辨率高度（从 `sod.ini` 的 `<Graphics.Resolution>` 读取）夹窗口 Y 值，防止 ZCN2 绘制越界。

## 3. Hook 列表

| Hook        | 地址                         | 类型     | 说明                 |
| ----------- | -------------------------- | ------ | ------------------ |
| BUILD hooks | 0x5F4503/0x5F3E75/0x5F491E | LoHook | 三界面生物信息窗口构建时执行布局调整 |
| DefProc     | 0x41B120                   | HiHook | 通用对话框消息处理（兜底调整）    |
| DescCreate  | 0x5F447F/0x5F3E54/0x5F489A | LoHook | 三界面描述控件 call 前改栈参数 |
| DlgInit     | 0x41AFA0                   | LoHook | 窗口 Y 吸附            |

## 4. 配置文件 MegaDesc.ini

```ini
[General]
Enabled=1

[Layout]
WindowHeight=487
Shift=13
DescY=248
TextHeight=207
TextWidth=200
DismissBtnMarginBottom=90
SpellBtnMarginBottom=132

[Images]
BackgroundPcx=bv_bgA.pcx
```

## 6. 源码结构

| 文件                                 | 说明                                      |
| ---------------------------------- | --------------------------------------- |
| `MegaDesc.cpp`                   | 主文件：全局 `_P`/`_PI` + 4 个 `#include`      |
| `modules/ConfigLog.inc.cpp`        | Config 结构体、INI 读取、日志系统                  |
| `modules/CreatureDialog.inc.cpp`   | 核心功能：窗口布局调整、按钮替换、BUILD/DefProc hooks    |
| `modules/PatchesAndImages.inc.cpp` | 窗口高度 exe patch、PCX24→Pcx8 加载/量化/合成、按钮替换 |
| `modules/Entry.inc.cpp`            | `StartPlugin()` 入口、`DllMain`、hook 注册    |
