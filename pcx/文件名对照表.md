# PCX 文件名对照表

MegaDesc 的运行时素材统一放在插件目录的 `pcx` 子目录，文件名统一使用 `bv_` 前缀。

## 命名限制

- **运行时会加载的素材文件名建议 ≤ 11 字符（含 `.pcx` 扩展名）**。
- 原因：HoMM3 / H3API 相关资源结构里常见资源名缓冲为 12 字节，按 C 字符串习惯需要预留结尾 `\0`，所以安全文件名长度按 11 字符控制。
- 本插件当前是自己从插件目录读取 PCX，不直接走游戏资源管理器；但为了避免后续改回资源系统、或复用 HoMM3 资源结构时踩坑，仍按 11 字符命名。
- `bv_prevA.pcx` / `bv_prevB.pcx` / `bv_prevC.pcx` 长度为 12 字符，**仅供人工预览/对照，不作为运行时素材名使用**。

## 命名规则

```text
bv_<类型><编号/状态>.pcx
```

| 前缀 | 含义 | 尺寸 | 运行时使用 |
|------|------|------|------------|
| `bv_bg` | 背景图 | 298×487 | 是 |
| `bv_prev` | 效果预览图 | 298×487 | 否，仅预览 |
| `bv_frm` | 按钮金框 | 50×42 / 50×38 | 是 |
| `bv_ok` | 确认按钮图标 | 48×40 | 是 |
| `bv_sp` | 魔法书按钮图标 | 48×36 | 是 |
| `bv_ds` | 解雇按钮图标 | 48×36 | 是 |

按钮图标末尾 `0`-`3` 对应 DEF 帧号：

| 后缀 | 状态 | 说明 |
|------|------|------|
| `0` | normal | 普通（DEF 帧 0，默认显示） |
| `1` | pressed | 按下（DEF 帧 1，鼠标点击） |
| `2` | highlight | 高亮（DEF 帧 2，预留） |
| `3` | disabled | 禁用（DEF 帧 3） |

## 文件清单

### 背景（298×487，24-bit / 3-plane PCX）

| 文件名 | 长度 | 说明 |
|--------|------|------|
| `bv_bgA.pcx` | 10 | 背景方案 A，可填入 `MegaDesc.ini` 的 `BackgroundPcx` |
| `bv_bgB.pcx` | 10 | 背景方案 B，可填入 `MegaDesc.ini` 的 `BackgroundPcx` |

### 效果预览（298×487，24-bit / 3-plane PCX，仅供参考）

| 文件名 | 长度 | 说明 |
|--------|------|------|
| `bv_prevA.pcx` | 12 | 效果预览 A；超过 11 字符，不建议作为运行时素材名 |
| `bv_prevB.pcx` | 12 | 效果预览 B；超过 11 字符，不建议作为运行时素材名 |
| `bv_prevC.pcx` | 12 | 效果预览 C；超过 11 字符，不建议作为运行时素材名 |
